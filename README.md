# 🔑 Cerrajería ACCTURA  

Este proyecto corresponde al desarrollo de la página web de **Cerrajería ACCTURA**, un negocio especializado en la fabricación, instalación y mantenimiento de puertas metálicas, escaleras metálicas y soluciones de seguridad para hogares, empresas y proyectos industriales.  

## 🌐 Objetivo del proyecto  
Diseñar y desarrollar una página web moderna y optimizada que represente los servicios de ACCTURA, facilite la visibilidad en línea y mejore la comunicación con clientes potenciales.  

## 🚀 Funcionalidades principales  
- Página de inicio con presentación de la empresa.  
- Sección de servicios (puertas metálicas, escaleras metálicas, estructuras).  
- Galería de trabajos realizados.  
- Información de contacto y ubicación en Google Maps.  
- Optimización SEO para buscadores.  

## 🛠️ Tecnologías utilizadas  
- **HTML5**, **CSS3**, **JavaScript** – Estructura, diseño y funciones interactivas.  
- **Google Business Profile** para mejorar posicionamiento local.  
- **Git/GitHub** para control de versiones.
